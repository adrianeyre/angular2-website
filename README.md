## <a name="Install">Installation</a>
* To clone the repo
```shell
$ git clone https://github.com/adrianeyre/angular2-website
$ cd angular2-website
```

* Setup Angular 2 Framework
```shell
npm install
```

* Run Angular 2 web framework
```
npm start
```

## <a name="Usage">Useage</a>
Navigate to ```localhost:3000```
